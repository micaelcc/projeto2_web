# projeto2_web
Projeto do módulo NodeJS da disciplina de Desenvolvimento de Software para Web.

## Instalar modulos
npm install

## Rodar migrations
npx sequelize db:migrate

## Rodar api
npm run dev

## Arquivo .env na raiz da aplicação

Exemplo: 

SECRET=exemplo<br/>
DB_NAME=exemplo<br/>
DB_PASS=exemplo<br/>
DB_HOST=exemplo<br/>
DB_USERNAME=exemplo<br/>
