# projeto2_web
Projeto do módulo NodeJS da disciplina de Desenvolvimento de Software para Web.

## Instalar modulos
npm install

## Rodar migrations
npx sequelize db:migrate

## Rodar api
npm run dev

## Arquivo .env na raiz da aplicação

Exemplo: 

SECRET=exemplo
DB_NAME=exemplo
DB_PASS=exemplo
DB_HOST=exemplo
DB_USERNAME=exemplo
